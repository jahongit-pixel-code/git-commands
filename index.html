git init: Yangi git loyihasi uchun bo'sh gitni ishga tushirish git clone: Ushbu buyruq orqali mavjud remote repositoryni olish. git add: Fayllarni staging area'ga qo'shish. git commit: Fayllarni lokal git repositoryga joylash. git push: O'zgarishlarni remote repositoryga jo'natish. git pull: O'zgarishlarni olish va merge qilish. git status: Hozirda loyiha holatini ko'rsatish. git branch: Branchlar haqida ma'lumot olish yoki yangi branch yaratish. git checkout -b komandasi Gitda yangi branch yaratish va unga o'tish uchun ishlatiladi. git checkout -d komandasi Gitda branchlarni o'chirish uchun ishlatiladi. Bu komanda o'zgartirilgan branchni o'chiradi. Afsuski, git checkout -d git merge: Branchlarni merge qilish. git log: Commitlar tarixini ko'rsatish. git remote: Remote repositorylarni boshqarish. git checkout: Fayllarni yangi versiyalarga o'tkazish.
